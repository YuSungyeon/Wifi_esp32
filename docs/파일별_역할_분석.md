# WiSLAR 파일별 역할 분석

이 저장소는 WiFi CSI 시계열을 입력으로 받아 샘플 단위 행동 인식(배경 + 6개 동작)을 수행하는 1D U-Net 기반 프로젝트입니다.

## 루트 파일

### `README.md`
- 프로젝트 개요(Temporal U-Net 기반 WiFi 행동 인식), 데이터/사전학습 가중치 다운로드 링크, 데이터 구조 설명을 담고 있습니다.
- 모델 구조 그림(`figs/unet1d.png`), AP 곡선(`figs/ap.png`), 예시 결과(`figs/result.png`)를 문서화합니다.
- 논문 인용 정보도 포함되어 있어 연구 재현/인용 출처 역할을 합니다.

### `train.py`
- 학습 실행 스크립트입니다.
- `data/train_data.mat`, `data/test_data.mat`를 읽어 `TensorDataset`/`DataLoader`를 구성하고, `UNet(n_classes=7)`을 GPU에서 학습합니다.
- 손실 함수는 `CrossEntropyLoss`, 옵티마이저는 `Adam`, 스케줄러는 `MultiStepLR`를 사용합니다.
- 매 epoch마다 train/test 정확도를 계산하고, 테스트 성능이 개선되면 `weights/*.pkl`로 모델을 저장합니다.
- 학습/평가 지표(`train_loss`, `test_loss`, `train_acc`, `test_acc`)를 `.mat` 파일로 저장합니다.

### `test.py`
- 학습된 모델 성능 평가 및 추론 결과 저장 스크립트입니다.
- `weights/Two96.253Test95.093.pkl` 가중치를 로드해 테스트 셋 정확도를 계산합니다.
- 예측 출력 텐서를 `out/Two96.253Test95.093.mat`로 저장합니다.
- `train.py`와 유사하게 `.mat` 데이터셋 로딩 및 배치 추론 흐름을 따릅니다.

## 모델 패키지 (`unet/`)

### `unet/__init__.py`
- 패키지 외부에서 `UNet` 클래스를 바로 import할 수 있도록 노출하는 진입점 파일입니다.

### `unet/unet_model.py`
- 전체 1D U-Net 네트워크 토폴로지를 정의합니다.
- 인코더(`inc`, `down1~4`)와 디코더(`up1~4`), 최종 분류 헤드(`outc`)를 조합해 forward 경로를 구성합니다.
- 기본 입력 채널 수 52(CSI 서브캐리어), 출력 클래스 수 7로 설정되어 있습니다.

### `unet/unet_parts.py`
- U-Net 구성 블록(재사용 가능한 모듈) 구현 파일입니다.
- `double_conv`: Conv1d + BatchNorm1d + ReLU를 2회 수행하는 기본 블록
- `down`: MaxPool1d + `double_conv`로 시계열 다운샘플링
- `up`: ConvTranspose1d(또는 보간) + skip connection concat + `double_conv`로 업샘플링
- `outconv`: 1x1 Conv1d로 클래스 로짓 생성

## 데이터/전처리 (`data/`)

### `data/segmentation.m`
- 원본 동작별 `.mat` 파일에서 CSI 진폭/위상을 읽고, 구간 라벨(시작/끝 시점)을 이용해 샘플 단위 레이블을 생성하는 MATLAB 전처리 스크립트입니다.
- `6in1.csv`의 파일명 및 구간 정보(`start`, `end`)를 기반으로 다음 데이터를 만듭니다.
  - `*_data_amp`, `*_data_pha`
  - `*_label_instance`, `*_label_mask`, `*_label_time`
- 5-fold 방식(인덱스 mod 5)으로 train/test를 분리해 `train_data.mat`, `test_data.mat`를 저장합니다.

### `data/6in1.csv`
- 각 샘플 파일명과 동작 구간 시작/종료 인덱스를 담은 메타데이터입니다.
- 전처리 스크립트(`segmentation.m`)가 레이블 생성 시 참조하는 기준 파일입니다.
